# sonnic-runner 🦔  - Pocket Edition

Um jogo de corrida infinita (Endless Runner) leve, moderno e totalmente otimizado para dispositivos móveis, desenvolvido puramente com **HTML5, CSS3 e JavaScript (Canvas)**.

---

## 📱 Sobre o Projeto

Este projeto foi construído de forma totalmente isolada e modular, focado na melhor experiência de jogabilidade arcade diretamente pelo navegador do celular. O design utiliza conceitos modernos de **Glassmorphism** (efeito de vidro fosco) na interface de usuário (HUD) para garantir um visual elegante e limpo.

### 🚀 Funcionalidades Principais
* **Controles por Toque (Swipe):** Mecânica responsiva deslizando o dedo para a esquerda ou direita para alternar entre as 3 pistas.
* **Sistema de Pontuação Progressiva:** A pontuação aumenta dinamicamente conforme a velocidade do jogo se eleva.
* **Coleta de Argolas:** Animações fluidas de rotação e efeitos de partículas ao coletar as clássicas argolas douradas.
* **Obstáculos Dinâmicos:** Geração aleatória de espinhos perigosos que exigem reflexos rápidos do jogador.
* **Interface Premium:** Telas de início e fim de jogo integradas perfeitamente, exibindo estatísticas finais de desempenho.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica e elemento `<canvas>` para renderização de alto desempenho.
* **CSS3:** Estilização responsiva, gradientes modernos e efeitos de desfoque de fundo (`backdrop-filter`).
* **JavaScript (ES6):** Motor lógico do jogo, gerenciamento de colisões (física por hipotenusa), sistema de partículas e manipulação do ciclo de animação (`requestAnimationFrame`).
